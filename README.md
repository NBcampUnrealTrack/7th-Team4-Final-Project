# 프로젝트 README

## 게임 컨셉

중세 풍의 어둡고 무거운 분위기를 배경으로, 악마와 언데드, 살덩이가 뭉쳐진 크리쳐들을 상대하는 액션 RPG.
휘황찬란한 판타지 느낌이 아닌, 어둡고 현실감 있는 중세 세계관을 지향한다.

## 재미 요소

### 전투
- **쫄몹 전투**: 디아블로 스타일의 핵앤슬래시 방식
- **보스 전투**: 로아(Lost Ark) 느낌의 패턴 기반 보스
  - 카운터 / 패링 / 구르기 등 회피 및 반격 메커니즘
  - 스태미나 관리 또는 스킬 시스템으로 난이도 조절

### 스킬 시스템
- 스킬 슬롯 자유 할당 (Q W E R / 1 2 3 4)
- 장착한 무기로 기본 전투 + 마법 등 다양한 스킬 자유 조합
- **직업 공통 패시브** 다수 + **직업 고유 패시브** 추가

### 소모품
- 보스전 포션 제한: 3~4개 제한 또는 쿨타임 적용으로 긴장감 부여

### 경제 / 파밍
- 몬스터 처치 시 아이템 드랍 (추후 수정 예정)
- 파밍한 아이템을 상점에서 판매
- 상점에서 장비 구매 후 **랜덤 옵션 뽑기** 시스템
